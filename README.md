# toxify

Hackable HTTP proxy to simulate system failure conditions. Built for [node.js](http://nodejs.org)/[io.js](https://iojs.org)

**Work in progress**

## Built-in poisons

- [x] Delay
- [_] Server error
- [_] Close socket
- [_] Bandwidth
- [_] Throttle
- [_] Debounce
- [_] Rate limit
- [_] Slow close
- [_] Slicer
