# toxify

Hackable HTTP proxy to simulate system failure conditions.
Built for [node.js](http://nodejs.org)/[io.js](https://iojs.org). Powered by [rocky](https://github.com/h2non/rocky)

**Work in progress**

## Built-in poisons

- [x] Delay
- [ ] Server error
- [ ] Close socket
- [ ] Bandwidth
- [ ] Throttle
- [ ] Debounce
- [ ] Rate limit
- [ ] Slow close
- [ ] Slicer
